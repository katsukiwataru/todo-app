# todo-app

## Usage

frontend setup

```sh
npm intall
```

```sh
npm run dev
```

- Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

```sh
npm build
```

- Builds the app for production to the `dist` folder.

database setup

```sh
docker-compose up
```

- Open [http://localhost:4466](http://localhost:4466) to view it in the browser.
